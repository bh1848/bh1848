## 📖 Research / Publications
- **MySQL과 Redis의 데이터 처리 성능 비교 평가**  
  *인터넷정보학회논문지 (Journal of Internet Computing and Services, JICS)*, Vol.25, No.3, pp. 35-41, 2024

  - **NoSQL 기반 캐싱 전략의 기술적 근거 제시**
